# CNN-CLONE

This project is a front-end clone of the CNN homepage, created to practice modern web development and improve UI/UX design skills. It focuses on recreating the layout and visual structure of CNN’s first page, including elements like the navigation bar, featured news section, and article previews. The goal of this project is to gain hands-on experience with building responsive and visually appealing web pages that resemble real-world websites.
The webpage is built using HTML5 for structure and CSS3 for styling, along with layout techniques such as Flexbox and Grid to achieve proper alignment and responsiveness. In some cases, JavaScript may be used to add interactivity. The design is made to be mobile-friendly so that it adapts well across different screen sizes.
This project helped in understanding how professional news websites organize content, manage layouts, and present information effectively. It also strengthens skills in responsive design, positioning, and component structuring.
To run this project locally, simply clone the repository, navigate to the project folder, and open the index.html file in any web browser. No additional setup or dependencies are required.
This project is created purely for educational purposes. All design inspiration, content, and trademarks belong to CNN. Future improvements may include adding dynamic content using APIs, enhancing accessibility, and introducing animations or additional features for a better user experience.
